Uma [lib](https://crates.io/crates/brasilapi) para a API do [BrasilAPI](https://github.com/BrasilAPI/BrasilAPI) (para o V)

# Features
- [x] **CEP (Zip code)**
- [X] **DDD**
- [X] **Bank**
- [x] **Corretoras**
- [x] **CNPJ**
- [X] **Feriados Nacionais**
- [ ] **Registros de domínios br**
- [ ] **IBGE**
- [ ] **Tabela FIPE**
- [ ] **ISBN**
- [ ] **Taxas**
